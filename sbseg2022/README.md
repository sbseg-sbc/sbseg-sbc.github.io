# SBSeg 2022 — Cópia estática para arquivamento

Cópia estática e navegável offline do site **SBSeg 2022** (https://sbseg2022.inf.ufsm.br),
capturada em 2026-06-16 para fins de arquivamento. A edição foi organizada pela Universidade
Federal de Santa Maria (UFSM).

## Como navegar

Abra **`index.html`** (na raiz desta pasta) em qualquer navegador — ele redireciona para a
homepage real. Alternativamente, abra direto `sbseg2022.inf.ufsm.br/index.html`.

Não é necessário servidor web; basta abrir os arquivos via `file://`.

## Estrutura

- `index.html` — redirecionamento de conveniência para a homepage.
- `sbseg2022.inf.ufsm.br/` — conteúdo do site (HTML, CSS, JS, imagens, fontes).
  - Páginas em português e na versão em inglês (`en/`).

## Características da cópia

- **Estática**: o site original é gerado com **Jekyll** e hospedado no GitHub Pages — já é
  estático por natureza; aqui há apenas o HTML final renderizado.
- **Links internos** foram reescritos para caminhos relativos — navegação 100% local.
- **Links externos** (UFSM, SBC, edições anteriores, redes sociais, anais, fontes do Google,
  Google Analytics) continuam apontando para a web — são navegação/recursos de saída e não
  fazem parte deste arquivo.

## Conteúdo capturado

- 61 páginas HTML (português + 17 em inglês)
- ~37 MB de conteúdo total
- 222 imagens, 8 arquivos de fonte, 3 CSS, 6 JS
